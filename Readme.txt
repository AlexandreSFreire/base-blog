# Blog Front-End

Este é um projeto de **blog front-end** desenvolvido utilizando **HTML**, **CSS** e **Bootstrap**. Ele oferece uma estrutura simples de blog, com layout responsivo e estilizado, apresentando artigos e conteúdos sobre desenvolvimento web.

## Instalação

Para rodar este projeto localmente, siga os passos abaixo:

1. Clone o repositório para sua máquina:

   git clone https://github.com/seu-usuario/nome-do-repositorio.git

2. Navegue até o diretório do projeto:

   cd nome-do-repositorio

Abra o arquivo index.html diretamente no seu navegador.

Uso
Este projeto é uma estrutura de blog que pode ser facilmente modificada para diferentes finalidades. Ele inclui uma navegação principal, uma seção de destaque com uma "hero image", e artigos recentes que abordam tópicos de desenvolvimento web como HTML, CSS e JavaScript.

Para modificar o conteúdo do blog, edite diretamente o arquivo index.html para adicionar novos posts, ajustar o layout ou modificar as categorias.

Estrutura do Projeto
O projeto está organizado da seguinte forma:

/ 
|-- index.html          # Arquivo principal HTML
|-- style.css       # Arquivo de estilos customizados (caso necessário)
|-- /img
|   |-- html.jpg        # Imagem usada no artigo HTML

Funcionalidades:

Responsivo: O layout é completamente responsivo, adaptando-se bem a diferentes tamanhos de tela, como mobile, tablet e desktop, graças ao Bootstrap.

Navegação simples: Um menu de navegação na parte superior facilita a movimentação entre diferentes seções do blog.

Artigos em destaque: A página inicial apresenta dois artigos principais com imagens e links para continuar a leitura.

<b>Seção "Sobre":</b> Um espaço dedicado para adicionar informações sobre o blog ou seus autores.

<b>Integração com Bootstrap:</b> A estrutura é fortemente baseada no framework Bootstrap 5 para facilitar o design e a responsividade.
